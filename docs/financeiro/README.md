## Resumo Executivo

- **Gateway:** Pagar.me (Stone Group) — proposta D15 aceita
- **Split:** AXON retém 9% + taxa de conveniência · Organizador recebe saldo líquido
- **Pix:** Recebimento imediato (~2 min) — método padrão recomendado
- **Cartão:** Opcional por evento, recebimento D+15, conveniência repassada ao comprador
- **Antecipação:** Automática embutida no modelo D+15 da Pagar.me
